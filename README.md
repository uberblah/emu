emu
===========================
A simple emulated processor

FEATURES
========
 - Execute Read/Writable Memory!
 - Memory Segmentation Based on Device Boundaries!
 - Simple Universal Device Interface!
 - Minimal Instruction Set!
 - Lots of Registers!
 - Typed Registers!

IMPLICATIONS
============
 - Great for Education!
 - Great for Videogames!
 - Easy to Learn!
 - Easy to Use!
