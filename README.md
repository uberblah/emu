emu
===========================
A simple emulated processor

FEATURES
========
 - execute read/writable memory!
 - move IP to a different memory device!
 - switch RAM to different devices!
 - simple device I/O structure!
 - minimal instruction set!
 - lots of registers!
 - typed registers!

IMPLICATIONS
============
 - great for education!
 - great for videogames!
 - easy to learn!
 - easy to use!
